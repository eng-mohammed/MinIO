# وصف المشروع

هذا المستودع يحتوي على مشروع برمجي يعتمد على .NET 10. الهدف من المشروع هو بناء تطبيق/خدمة قابلة للتشغيل باستخدام أدوات .NET ومناسبة للتطوير المحلي والنشر.

## المتطلبات

- .NET 10 SDK مثبت
- Microsoft Visual Studio 2026 أو أي محرر يدعم .NET
- PowerShell (لتشغيل الأوامر النصية)
- (اختياري) Docker إذا كان المشروع يتطلب حاويات

## الإعداد والبدء

1. استنساخ المستودع:

   `git clone <repo-url>`

2. الانتقال إلى مجلد المشروع:

   `cd <repo-folder>`

3. استعادة الحزم وبناء المشروع:

   `dotnet restore`

   `dotnet build`

4. تشغيل المشروع (أحد الخيارات):

   - من خلال Visual Studio: افتح ملف الحل `.sln` واضغط F5 أو اختر "Start Debugging".
   - من خلال الطرفية:

     `dotnet run --project path/to/YourProject.csproj`

## الإعدادات البيئية

- إعدادات البيئة والتكوين موجد في `appsettings.json` أو متغيرات البيئة. قم بتعديل القيم ذات الصلة مثل سلاسل الاتصال (connection strings)، مفاتيح API، أو إعدادات التخزين عند الحاجة.

## تشغيل الاختبارات

لتشغيل اختبارات الوحدة استخدم:

`dotnet test`

أو قم بتشغيل اختبار محدد من خلال Test Explorer في Visual Studio.

## النشر

اتبع خطوات النشر الخاصة بالمنصة المستهدفة (Azure, Docker, on-premise). مثال للنشر كحزمة ذاتية الاستضافة:

`dotnet publish -c Release -o ./publish`
